## Who we are

NextStage-SY is a software agency founded by **Abdullah Khatib** and **Mohammad Tarabishi**. We run two engines side by side:

- **Client services** — we design, build, and ship software for clients.
- **Our own products** — we build and launch products of our own.

## Services

- **Product engineering** — end-to-end design and delivery of web applications.
- **Full-stack development** — .NET / C# backends with Angular front-ends.
- **API & backend systems** — layered, domain-driven architectures built to scale.
- **Cloud & DevOps** — containerized delivery with Docker and CI/CD pipelines.
- **Bilingual apps** — Arabic & English (RTL) from day one, built for MENA and diaspora audiences.

## Our products

**[Athar](https://github.com/NextStage-SY/Athar-Web)** (أثر) — a community-sustained Islamic knowledge platform, powered by donations and volunteers rather than subscriptions. Open and public — explore the code in **[Athar-Web](https://github.com/NextStage-SY/Athar-Web)**.

*More products are in the works.*

## How we build

- **Backend** — .NET 10 / C# on the [ABP Framework](https://abp.io) (layered, DDD)
- **Frontend** — Angular with the LeptonX Lite theme
- **Data** — PostgreSQL
- **Delivery** — Docker + GitHub Actions CI
